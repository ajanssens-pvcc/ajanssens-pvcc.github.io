modular implementation A module is a distinct assembly of components that can be easily added, removed or replaced in a larger system. This implementation process involves swapping out multiple parts to see what makes the system work best.
Unifed Modeling Language (UML) A general-purpose modeling language that is intended to provide a standard way to visualize the design of a system.
