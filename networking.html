Combining Networks
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
  • A device that passes a signal between two buses.
  • A device that passes a signal between two buses based on destination address.
  • A bridge that connects multiple buses.
  • A device that allows connections between two networks while allowing each network to maintain its unique internal characteristics.
  • A device that acts as a router but whose main purpose is to communicate between the external and internal network.

Internet Protocols
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol - an application for listing and copying files across the network.
  • Hyper Text Transfer Protocol - an application for handling webpages.
  • Simple Mail Transfer Protocol - an application that defines the way two computers on a network interact when transmitting an email message.

Internet Four-Layer Software
  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer
  • Software units such as clients and servers that use Internet communication to carry out tasks.
  • The devices and protocols responsible for sending and receiving messages over the Internet.
  • The software residing on Internet routers that decides in which direction a packet is sent.
  • The router in charge of receiving and transmitting packets.
Internet is:
  • Packet-Switching Network
  • Fault Tolerant
  • Redundant
  • A method of grouping data into packets that are transmitted over a digital network. Packets are made of a header and a payload
  • A fault-tolerant system is one that can experience failure (or multiple failures) in its components, but still continue operating properly.
  • A communications pathway that has additional links to connect all nodes in case one link goes down.

  • Spyware
